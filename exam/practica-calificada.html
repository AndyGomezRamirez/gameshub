PARTE 2: Componentes y Navegación

40 puntos

2.1 Crear Componente de Estadísticas (20 puntos)

Comando: ng generate component components/estadisticas --standalone

El componente debe mostrar:

  • Total de juegos en la plataforma
  • Cantidad de juegos gratuitos vs de pago
  • Juego con mejor rating (nombre y rating)
  • Promedio de precios (solo juegos de pago)
Requisitos técnicos:
  • Inyecte y use JuegosDataService
  • Suscriba correctamente a los Observables
  • Use pipes: currency para precios, number para decimales
  • Implemente ngOnInit para cargar datos

2.2 Configurar Navegación (20 puntos)

En app.routes.ts:

{
  path: 'estadisticas',
  component: EstadisticasComponent
}

En app.component.html:

  • Agregue enlace después de "Categorías"
  • Texto: "Estadísticas"
  • Icono: bi-graph-up
  • Use routerLink y routerLinkActive

PARTE 3: Servicios y Funcionalidad

25 puntos

3.1 Extender Servicio de Juegos (15 puntos)

Archivo: src/app/services/juegos-data.service.ts

Agregue estos métodos:

Método 1: getJuegosPorPrecio
  • Parámetros: min: number, max: number
  • Retorna: Observable<Juego[]>
  • Filtra juegos en el rango de precio especificado
Método 2: getEstadisticas
  • Sin parámetros
  • Retorna: Observable<Estadisticas>
  • Objeto con: totalJuegos, juegosGratis, juegosPago, mejorRating, promedioPrecio

3.2 Implementar Ordenamiento (10 puntos)

En lista-juegos.component, agregue selector de ordenamiento:

Opciones de ordenamiento:
  • Nombre: A-Z / Z-A
  • Precio: Menor a Mayor / Mayor a Menor
  • Rating: Mejor a Peor

Ubicación: Debajo de la barra de búsqueda
